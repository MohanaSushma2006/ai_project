
# Placement Predictor AI Project

## Description
This project predicts whether a student will be placed or not using Machine Learning.

## Features
- Input: CGPA, Skills, Projects
- Output: Placement Prediction
- Real-time prediction using Flask API

## Technologies Used
- Python
- Flask
- Machine Learning (scikit-learn)
- HTML, CSS

## How to Run
1. Run train_model.py
2. Run app.py
3. Open index.html
4. Enter values and click Predict

## Output
- 1 → Placed
- 0 → Not Placed
